WINDOWS

    MÍNIMOS:
  • Requer um processador e sistema operacional de 64 bits
  • SO: Windows 7 ou mais recente (64-bit)
  • Processador: 2.8 Ghz Quad Core CPU
  • Memória: 8 GB de RAM;
  • Placa de vídeo: 2 GB de memoria dedicada
  • DirectX: Versão 11
  • Rede: Conexão de internet banda larga
  • Armazenamento: 15 GB de espaço disponível
  • Placa de som: Compatível com DirectX
  • Outras observações: Executar um Servidor Dedicado e um Cliente no mesmo computador duplicará os requisitos de memória RAM. Além disso, versões futuras podem exigir mais espaço no disco rígido.
    RECOMENDADO:
  • Requer um processador e sistema operacional de 64 bits
  • SO: Windows 7 ou mais recente (64-bit)
  • Processador: 3.2 Ghz Quad Core CPU ou melhor
  • Memória: 12 GB de RAM
  • Placa de vídeo: 4 GB de memoria dedicada
  • DirectX: Versão 11
